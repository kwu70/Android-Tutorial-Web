Genymotion

Genymotion Emulator

  • Downloading Genymotion
  • Running Installation for Windows
  • Running Installation for Mac
  • Installing Genymotion Plugins and Setting the Emulator

Genymotion is a fast Android Emulator that is free for private uses. It will allow you to test your Android application much faster than most other emulators and comes highly recommended. However, to run the Android application in Genymotion, you have to start Genymotion first, then run the application as the Genymotion emulator.

Installing Genymotion

To download Genymotion:

Visit: https://www.genymotion.com/#!/download

An account is required to download Genymotion. Creating an account is free, so go ahead a create one. After the account is created, go to the download page and select the download with VirtualBox. If you already have the VirtualBox virtual machine, then go ahead and download the one without VirtualBox, since VirtualBox is required for run the emulator.

Open the .exe file. Install Genymotion:

Go through the intallation wizard to finish installation:

After Genymotion is finished installing, the VirtualBox installation wizard should automatically open up.

Select VirtualBox installation option:

Select the default setting.

Running Installation for Mac

Open the .dmg file and drag the Genymotion and Genymotion Shell to your application folder:

Installing Genymotion Plugins and Setting the Emulator

The Genymotion plugin is required for running the Android application in the emulator.

Open up Android Studio. On the quick start screen, select configure:

Select plugins:

Search for genymotion on the search bar and download:

Set up an emulator:

After installing the plugins, there will be a Genymotion tool bar at the top right. To create a new Android emulator, select "New" and Genymotion will open up with options for the new emulator download. Alternatively, if the emulator is already created, then just select the emulator and press "Start".

Select the emulator you need and download:

To run the application in the newly downloaded emulator:

Choose the downloaded emulator and select "Start". When you run the Android application you created, a option will popup asaking you how you want to run the application. Choose the open emulator from Genymotion.
