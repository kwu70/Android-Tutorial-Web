What is an Activity?

An Activity represents a single screen with a user interface. The user can interact with this screen to do something such as enter text, take a photo, look at data.

Applications usually consist of multiple Activities (one per user interface screen).

One of the activities is specified as the main activity, which is the first activity that appears when the application is launched.

When you go from one Activity to a new Activity, the old Activity is stopped, but is saved in the back stack. (You can press the back button to go back to the old Activity).

For more information:
Activity API: http://developer.android.com/reference/android/app/Activity.html
Activity Developer Guide: http://developer.android.com/guide/components/activities.html

Creating a New Activity

To create a new Activity you need to right click the package you want the Activity in go to New->Activity then choose the Activity template you want to create (We will just be using a Blank Activity).


Now you will be presented with a Dialog which looks something like this:

Name your Activity whatever you want (We named ours MyNewActivity) Most of the fields will fill in and change automatically as you change the Activity Name (This is a good thing so that the files can be properly linked and synchronized).

Now Android Studio will create your new Activity as well as an xml file (activity_new.xml). The MyNewAcitivity.java and activity_new.xml are linked together by this line of autogenerated code: setContentView(R.layout.activity_my_new);


This is the layout (User Interface) for the Activity (discussed more in-depth in the User Interface Section of this tutorial).

The onCreate() method is called when this Activity is first created. You should initialize the essential components of your Activity.

You must implement the onCreate() method to perform basic startup logic. This logic should only happen once on the creation of the application. The main uses for onCreate() are generally:

Declaring User Interface variables
Configuring User Interface variables (Listeners, Initialization etc.)
Defining variables

Going from One Activity to Another

To go from one Activity to another Activity (example: LoginActivity to UserHomeActivity) you can use an Intent

Intents can be used to start a new Activity by passing an instance of the Activity to an Intent.

For a more indepth look at Intents:
Intent API: http://developer.android.com/reference/android/content/Intent.html
Intent Developer Guide: http://developer.android.com/guide/components/intents-filters.html

The following is an example that goes from this Activity to the UserHomeActivity.

Intent userHomeIntent = new Intent(this, UserHomeActivity.class);
startActivity(userHomeIntent);

This is usually used in conjunction with a Button. So the user presses a button to go to the next Activity and the above code is run to go to the next Activity. Please see our guide on Buttons in the Android UI section of this tutorial.

If you are going from one Activity to a new Activity and wish to send some data that the new Activity can add these to the Intent using the putExtra method.

putExtra allows you to add some data to the Intent which can then be retrieved in the new Activity. There are many different putExtra methods which allow for different data types.

Here are some you might find useful:

public Intent putExtra(String name, int value)
public Intent putExtra(String name, int[] value)
public Intent putExtra(String name, double value)
public Intent putExtra(String name, double[] value)
public Intent putExtra(String name, String value)
public Intent putExtra(String name, String[] value)

These are just a few examples, but most of the basic data types are allowed in the putExtra method. A full list of data types allowed is in the Android API for Intents.

Parameters:
name - The name of the extra data with package prefix
value - the data value
Returns - the same Intent

The following example puts a username and password into a String array and passes it to the Intent.

String[] userPass = new String[] {“username”, “password”};
Intent userHomeIntent = new Intent(this, UserHomeActivity.class);
userHomeIntent.putExtra(“user_pass”, userPass);

Now when we go to the userHomeActivity using the userHomeIntent we can retrieve the data using the following code:

Intent intent = getIntent();
String[] userPass = intent.getExtras().getStringArray(“user_pass”);
