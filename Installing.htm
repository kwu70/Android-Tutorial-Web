Installing Android Studio on Windows

  • Installing Java Developer Kit (JDK)
  • Installing Android Studio

To develop for Android you need to get an IDE for Android. Android Studio is arguably the best Android IDE since it was made by the creators of Android. This tutorial will be using Android Studio so it is highly recommended you get Android Studio.

Installing Java Developer Kit (JDK)

  • Go to http://www.oracle.com/technetwork/java/index.html
  • Choose Java SE under Top Downloads
  • Choose JDK Download
  • Choose the most recent download (appropriate OS)
  • Setting up environment variable
    • Window 7
      • Choose “Properties” of the Computer
      • Click “Advance system settings”
      • Click “Environment Variables...”
        • Create Variable
          • Variable name: JAVA_HOME
          • Variable Value: C:\Program Files\Java\jdk1.8.0_31 (location of the JDK folder)
          • If you don't know where JDK folder is located
            • Go to “Program Files” directory
            • Search for Java
            • Then you should find the JDK folder there
          • Be sure to choose JDK folder (NOT JRE folder)

Installing Android Studio

  • Go to http://developer.android.com/sdk/index.html
  • Download Android Studio Installer
    • Click "Download Android Studio for Windows"
    • Agree on the terms and conditions
    • Click "Download Android Studio for Windows"
  • Click “Next” on Welcome screen
  • Click “Next” on Choose Components screen
  • Click “I Agree” on License Agreement screen
  • On Configuration Settings (Install Locations) Screen
    • Recommended to leave “Android Studio Installation Location” default
    • Change the location of the “Android SDK Installation”.
      Note: This should be placed somewhere other than under the Android Studio installation directory.
    • Then click “Next”
  • Click “Next” to accept the default setting on Configuration Setting (Emulator Setup) screen
  • Click “Install”
  • This will take a few minutes to do the primary installation.
  • When you see the screen which indicates the installation is complete, click “Next”
  • Finally, click “Finish” to start the Android Studio.
  • If this is the first time installing Android Studio. the “Setup Wizard” screen will appear to download additional SDK components

Installing Android Studio on Mac

  • Installing Java Developer Kit (JDK)
  • Installing Android Studio

To develop for Android you need to get an IDE for Android. Android Studio is arguably the best Android IDE since it was made by the creators of Android. This tutorial will be using Android Studio so it is highly recommended you get Android Studio.

Before installing Android Studio on Mac OS X, make sure that your computer has the recommended versions of the operating system and of Java Developer Kit (JDK). You can check your Mac's operating system by clicking the Apple Menu and choosing About This Mac:

You need Version 10.8.5 or higher:

To check if you have JDK installed and it's version go to Terminal and type java -version:

We recommend using the most recent version, JDK 8:

Installing the Java Developer Kit (JDK)

In order to develop Android apps, you will need to install the JDK. This is more than the Java Runtime, which might already be installed on your computer. If you need to install JDK or update to the most recent version, go to your web browser and navigate to the web page at http://www.oracle.com/technetwork/java/index.html. From this page, click the Java SE link under Top Downloads:

Then click the download link under JDK:

After accepting the License Agreement, you will be able to download the version of the JDK for Mac OS X 10.8 (Mountain Lion) or later.

Downloading and installing Android Studio

Once you have the right version of JDK, you can install Android Studio. Go to your web browser and navigate to the web page at http://developer.android.com/sdk/index.html. Click the download link:

The installer is downloaded as a *.dmg file. To start the installation, double-click it to open it:

Then, drag-and-drop Android Studio into the Applications folder:

Once the file has been copied over, double-click on Applications. Double-click Android Studio.

When it is complete, the software is installed and ready to use,However, you will probably want to install extra packages to make Android Studio complete. When you open Android Studio, you should be presented with the "Welcome to Android Studio" window. Click Configure:

Click SDK Manager:

Check the packages you want to install and click "Install packages..." button:

It is recommended that you download:

  • Android SDK Tools
  • Android SDK Platform Tools
  • Android SDK Build Tools
  • A System Image for the emulator, (like ARM EABI v7a System Image)

The installation will probably take a few minutes. Once the installation is complete, you can use Android Studio.

You can find the latest system requirements for Android Studio at the http://developer.android.com/sdk/index.html#Requirements

To read more about installing Android Studio refer to the http://developer.android.com/sdk/installing/index.html?pkg=studio
