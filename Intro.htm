Overview

What is Android?

Android is the Linux-based mobile operation system developed by Google. Android is the most popular mobile platform and growing fast. It is run in hundreds of millions of mobile devices around the world. Android offers a unified approach to application development for mobile devices which means developers need only develop for Android, and their applications should be able to run on different devices powered by android, as well as open marketplace for distributing to them instantly.
